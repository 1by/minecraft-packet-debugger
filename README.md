# minecraft-packet-debugger
![screenshot](https://i.imgur.com/wFDjg2Q.png)

## Alpha usage:
```sh
git clone git@github.com:wvffle/minecraft-packet-debugger.git
cd minecraft-packet-debugger

yarn install
node index.js

# Now navigate to http://localhost:3000/
```
